# MiCS5524 — CO, Alcohol & VOC Gas Sensor

**Status: Deferred — not yet wired.**

## Basics
- **Interface:** Analog (not I2C)
- **Pins:** Ao (analog out), En! (enable, active low), GND, 5V
- **Detects:** CO (1–1000 ppm), Ethanol (10–500 ppm), Ammonia (1–500 ppm), H2, CH4, VOCs
- **No library needed** — raw `analogRead()`

## Connection Plan
```
MiCS5524    →  Nano ESP32
Ao          →  A0 (ADC1, WiFi-safe)
En!         →  GND (always enabled) or GPIO for power control
GND         →  GND
5V          →  5V (VBUS)
```

## Reading
```cpp
int raw = analogRead(A0);              // 0–4095 (12-bit)
float voltage = (raw / 4095.0) * 3.3;  // 0–3.3V
```

## Quirks
- **Non-linear response** — voltage-to-ppm requires per-gas calibration curves
- Best used as relative indicator ("gas detected") rather than precise measurement
- **Heater warmup:** 1 sec minimum, 3 min in clean air for baseline
- **First use:** 24-hour burn-in for stable baseline
- **ESP32 ADC2 pins don't work with WiFi** — must use ADC1 (A0 is safe)
- Cannot distinguish between different gases
- 5V sensor output into 3.3V ADC — verify breakout board has level shifting
